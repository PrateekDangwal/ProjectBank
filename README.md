## 📌 Project Overview

This project is a **console-based Bank Account Management System** built using Java. It demonstrates core Object-Oriented Programming (OOP) concepts by simulating real-world banking operations such as deposits, withdrawals, transfers, interest calculation, and overdraft handling.

---

## 🚀 Features

* 💰 Deposit and Withdraw functionality
* 🔁 Transfer money between accounts
* 📈 Savings Account with interest calculation
* 🏦 Current Account with overdraft facility
* 🔒 Encapsulation for secure data handling
* ♻️ Code reuse using inheritance
* 🔁 Method overriding for custom behavior

---

## 🧠 OOP Concepts Used

### 1. Classes & Objects

* Created multiple classes: `BankAccount`, `SavingAccount`, `CurrentAccount`
* Used objects to simulate real bank accounts

---

### 2. Encapsulation

* Used `private` variables (like `balance`)
* Accessed data through methods (`getBalance()`, `updateBalance()`)

---

### 3. Inheritance

* `SavingAccount` and `CurrentAccount` extend `BankAccount`
* Reused common functionalities like deposit, transfer

---

### 4. Polymorphism (Method Overriding)

* Overrode `withdraw()` in `CurrentAccount`
* Implemented custom logic using overdraft limit
